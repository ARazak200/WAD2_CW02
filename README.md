## How to run the site

1. Clone this Repo
 ```bash
 git clone https://github.com/ARazak200/WAD2_CW02.git
 ```

2. Change into the root directory of the project 
```bash
cd WAD2_CW02
```

4. Place the data into the databases

```bash
npm start data 
```
6. run the run the site
  ```bash
  npm index
  ```
